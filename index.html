Intro
Here was originally supposed to be a webdev portfolio, but then I came across some delicious indie personal sites and I was like, "Woah, that's so cool!"
So here we are now. Before I started designing this site, I was like, "What kind of aesthetic am I going for?" Then I saw those personal sites where it was kind of a retro vintage 1990s vibe (specifically the Windows95 vibe), and I also wanted to make it super interactive, and needless to say I settled for it.
I wasn't around here for the era of personal websites (I'm a 2010s kid), but I love how expressive and creative they are, it's like going through someone's scrapbook. I find it sorta difficult to express myself with words and even art, so I hope this website can do that well because really, I don't know.
Another thing I must add, I saw a playthrough of this horror game called "Vile: Exhumed" (not on steam atm), and the whole desktop UI thing was probably the reason why this idea got into my head in the first place lol. More about the game in the 'yapping' section btw.
Development
I've made a portfolio before (that will never see the light of day lol), and the first mistake I made was not planning out the design. I hopped onto Figma, found a community resource called Windows 95 UI Kit by yarin bash, found a few references from GUIdebook and then I layed out the design.
After that I coded the bare HTML, after that I started styling it with CSS. I suck at CSS grid so I had help with the CSS Grid Generator by sarah_edo and everything became soooooo much easier. A tip if you want less confusion with CSS - never have 5 different button classes with very vague names (speaking from experience???) lol.
And then it was time for Javascript. My hands get really tired when I code so pure vanilla JS was a no-no, henceforth I made use of jQuery. I searched up a bunch of tutorials on how to make the windows draggable (people who use pure CSS are masochists), and that was when I found jQuery UI. Literally my saviour. It's kinda embarassing to admit that I had a hard time putting Jquery UI into this project, but I found a tutorial from 4 years ago that was still relevant.
Then came the sound design part. Got my hands on a free Sound Effect pack from SwishSwoosh. Ahhhh I love clickety clackety sounds, they scratch my brain the right way.
And that's basically it! I'll (probably) add more stuff in the future, and update regularly.
